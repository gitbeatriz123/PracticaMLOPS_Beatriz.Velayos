# Práctica FastAPI + HuggingFace

## Ejecución local

1. Crea un entorno virtual:
   python3 -m venv venv
   source venv/bin/activate

2. Instala dependencias:
   pip install -r requirements.txt

3. Lanza el servidor:
   uvicorn main:app --reload

4. Accede a la API en http://localhost:8000/docs

## Archivos entregados

- main.py
- requirements.txt
- Carpeta capturas/
